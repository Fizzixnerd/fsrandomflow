Getting Started

Open both the FsRandomFlow namespace and the FsRandomFlow.RVar module. Once you have done this, the randomly builder will let you compose in existing RVarT values of various types to build ones of your own. To run an RVarT value, use one of runrvarT or runrvarIOT, supplying a seed yourself or getting a seed from the current time respectively.
