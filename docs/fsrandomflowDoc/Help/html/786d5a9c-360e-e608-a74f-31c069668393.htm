RVar Methods

The RVar type exposes the following members.

Methods
  NameDescription
applyT, U
Builds a new random variable that applies results sampled from an RVar<'T> to functions sampled from an RVar<'T -> 'U>.
concatMapT, U
Builds a new random variable that uses random results sampled from an RVar<'T> to produce random variables of type RVar<'U> using the provided function. Those random variables are then sampled by the same random generator.
constantT
Creates a random variable that, when sampled, returns one and only one value, not polling the underlying random generator.
Exponential
A random variable that, when sampled, gets a double from an exponential distribution with a given inverse scale.
filterT
Creates a new random variable that, when sampled, samples another random variable repeatedly until the sampled value satisfies a given predicate.
filterRandomlyT
Creates a new random variable that, when sampled, samples another random variable repeatedly until the sampled value passes a randomized test.
mapT, U
Builds a new random variable of type RVar<'U> by mapping a given function 'T -> 'U over a random variable of type RVar<'T>
Normal
A random variable that, when sampled gets a double from a normal distribution with the given mean and standard deviation
oneOfT
Creates a random variable that, when sampled, returns one of the given values, each with the same probabilty.
oneOfWeightedT
Creates a random variable, that, when sampled, returns one possibility out of many, with given weights. Weights will be ignored if they are not greater than 0.
probability
A random variable that, when sampled, yields true with the given probability.
RandomlySignedDouble
A random variable that, when sampled, randomly flips the sign of a double and returns the result. The chance of either result is equal.
RandomlySignedFloat
A random variable that, when sampled, randomly flips the sign of a double and returns the result. The chance of either result is equal.
RandomlySignedInt
A random variable that, when sampled, randomly flips the sign of an integer and returns the result. The chance of either result is equal.
repeatT
Creates a random variable that streams the output of another random variable with a branched random generator. Polling the enumerator samples from it with the branched generator.
runrvarT
Run a random variable using a given seed.
runrvarIOT
Run a random variable using some random seed chosen non-deterministically.
sequenceT
Creates a random variable from a sequence of random variables, that, when sampled, samples each random variable given in order.
sequenceParallelT
Creates a random variable that, when sampled, samples each of the provided random variables in parallel.
sequenceStreamingT
Creates a random variable that, when sampled, produces an enumerator that samples each provided random variable in order using a branched random generator.
shuffleT
Shuffles a finite input. A new array is allocated to hold the output.
tableColumnT
Creates a new random variable that, when sampled, retrieves a random column from a given 2D array.
tableColumnWiseT
Creates a new random variable that, when sampled, retrieves a random element from each row in a given 2D array.
tableRowT
Creates a new random variable that, when sampled, retrieves a random row from a given 2D array.
tableRowWiseT
Creates a new random variable that, when sampled, retrieves a random element from each column in a given 2D array.
takeT
Creates a random variable that, when sampled, samples another random variable a given number of times, and returns an array of the samples
takeParallelT
Creates a random variable that, when sampled, samples another random variable a given number of times in parallel, and returns an array of the samples.
UniformInt
A random variable that, when sampled, yields an integer in between the two given bounds, inclusive. All integers within these bounds have an equal chance of being selected. The bounds can be given in any order.
UniformInterval
A random variable that, when sampled, yields an double in between the two given bounds. All double within these bounds have an equal chance of being selected. The bounds are given lowest first, then highest. You must specify if the bounds are open or closed.
UniformIntervalClosed
A random variable that, when sampled, yields an double in between the two given bounds, inclusive. All doubles within these bounds have an equal chance of being selected. The bounds can be given in any order.
UniformIntervalOpen
A random variable that, when sampled, yields an double in between the two given bounds, exclusive. All doubles within these bounds have an equal chance of being selected. The bounds can be given in any order.
UniformZeroAndUpBelow
A random variable that, when sampled, yields a non-negative integer less than n.
unionT
Create a random variable that samples from some number of other random variables with equal chance.
unionWeightedT
Create a random variable that samples from some number of other random variables with given chances.
Weibull
A random variable that, when sampled, gets a double from an Weibull distribution with given parameters, giving times to failure.
zipT, U
Creates a random variable out of two random variables, which, when sampled, returns samples from both variables in a pair.
zip3T, U, V
Creates a random variable out of three random variables, which, when sampled, returns samples from all three variables in a triple.
